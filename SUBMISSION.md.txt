# Submission

## Project Overview

This is a full stack document editor application with document creation, editing, upload, and sharing features.

---

## Features Implemented

- Document creation
- Document editing
- Rename document
- Save document
- Upload file (.txt)
- Share document
- Persistence using SQLite

---

## Tech Stack

Frontend:
- React
- TipTap

Backend:
- Node.js
- Express

Database:
- SQLite

---

## How to Run

Backend:

cd backend  
npm install  
node server.js  

Frontend:

cd frontend  
npm install  
npm start  

---
