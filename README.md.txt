# Document Editor - Full Stack Application

## Overview

This is a full stack document editor application that allows users to:

- Create documents
- Edit documents
- Rename documents
- Upload files
- Share documents
- Save and reopen documents

---

## Tech Stack

Frontend:
- React
- TipTap Editor

Backend:
- Node.js
- Express

Database:
- SQLite

---

## Features

### Document Creation

- Create new document
- Rename document
- Edit document
- Save document

### Rich Text Editor

- Bold
- Italic
- Headings
- Lists

### File Upload

Supported file types:

- .txt

Users can upload files and convert to document.

### Sharing

- Share document via email
- Owner access
- Shared documents

### Persistence

- SQLite database
- Documents saved after refresh
